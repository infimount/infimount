# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial hybrid storage browser with local filesystem support
- S3, Azure Blob, GCS, and WebDAV backend configurations
- File browsing with grid and list views
- File preview panel with image/text support
- Drag-and-drop file uploads
- Storage sidebar with multiple source management
- Cross-platform desktop app (Linux, macOS, Windows)

### Infrastructure
- GitHub Actions CI pipeline
- Multi-platform release workflow
- Governance and maintainer documentation
