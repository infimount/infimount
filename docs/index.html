Linux

Use a native package for your distro, or the AppImage for a portable install.

Download DEB.deb Download RPM.rpm Download AppImage.AppImage

macOS

Install using the DMG, or use the Homebrew cask if you prefer terminal-based upgrades.

Download DMG.dmg Homebrew caskbrew

Windows

Use the MSI installer or NSIS setup executable. Unsigned releases can trigger SmartScreen.

Download MSI.msi Download Setup.exe

Install with Homebrew

brew tap infimount/infimount
brew install infimount
brew install --cask infimount

Linux uses the formula. macOS can use the cask. Upgrade with brew update and brew upgrade.

First run notes

  • Linux AppImage: run chmod +x Infimount-*.AppImage before launch.
  • macOS: drag the app to Applications, then right-click and choose Open on first launch if Gatekeeper warns.
  • Windows: if SmartScreen appears, choose More info then Run anyway.
